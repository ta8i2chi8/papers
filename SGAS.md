# SGAS: Sequential Greedy Architecture Search

アーキテクチャ設計は、深層学習を成功させるための重要な要素となっています。最近のニューラル・アーキテクチャの自動探索（NAS）の進歩には、大きな期待が寄せられています。しかし、発見されたアーキテクチャーは、最終的な評価では一般化できないことが多い。探索段階での検証精度が高いアーキテクチャーは、評価でのパフォーマンスが悪くなることがあります（図1参照）。この問題を解決するために、私たちは、ニューラル・アーキテクチャ探索の効率的な手法であるSequential Greedy Architecture Search (SGAS)を導入しました。SGASは、探索手順をサブプロブレムに分割することで、候補となる演算の選択と刈り込みを欲張りな方法で行う。我々はSGASを畳み込みニューラルネットワーク(CNN)とグラフ畳み込みネットワーク(GCN)のアーキテクチャの探索に適用した。大規模な実験により、SGASは、画像分類、点群分類、タンパク質-タンパク質相互作用グラフのノード分類などのタスクに対して、最小限の計算コストで最先端のアーキテクチャを見つけることができることがわかった。

## 探索と評価の相関
* 既存のNASアルゴリズムの一般的なパイプライン[66, 35]は、探索フェーズと評価フェーズの2つのステージを含んでいる。計算オーバーヘッドを削減するために、先行研究[66, 35]では、まず、小規模なプロキシデータセット上で、軽量なプロキシモデルを用いて、あらかじめ定義された探索空間を探索する。そして，最適なアーキテクチャーセル／エンコーディングが得られた後，最終的なアーキテクチャーを構築し，ターゲットデータセット上でゼロから学習させる．この場合、評価時の真の性能が探索段階で推測できることが前提となる。しかし、探索フェーズと評価フェーズでは、データセット、ハイパーパラメータ、ネットワーク・アーキテクチャが異なるため、この仮定は通常成り立ちません。
* 理想的なNA法は，検索評価のKendall τ係数が高いことです。（Kendall τ係数：相関性を測定する指標）DARTSを例にとり、そのKendall τを図1に示します。これは、検索段階と評価段階のランキングの間で計算されます。この順位は、CIFAR-10データセットでの10回の異なる実行後の検証精度と最終評価精度に応じて決定される。DARTSのKendall τ係数は、1次版と2次版でそれぞれ0.16と-0.29しかありません。したがって、以下のような信頼性の高い予測を行うことはできません。

## Sequential Greedy Architecture Search
* 縮退した探索・評価相関問題を緩和するためには，(1)探索段階と評価段階の間の不一致と，(2)重み共有の負の効果を低減することが重要な側面である．我々は、二段階最適化（式1、2）をシーケンシャルに貪欲に解くことで、モデルの不一致と重み付けを段階的に削減することを提案する。
* DARTSに基づいた手法[35, 9, 61]は、緩和された問題を完全に解き、最後に選択されたすべての操作を得ることができます。完全な問題を直接解く代わりに、我々はそれをサブ問題に分割し、貪欲なアルゴリズムでそれらを逐次的に解く。


## まとめ
DARTSの探索戦略にGreedy法を取り入れたもの。

NASの二段階最適化問題は、エッジの重要性、選択の確実性、選択の安定性を考慮したヒューリスティックな基準を用いて、貪欲に解決される。
